# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-06

### Added
- Initial release of OpenClaw Channel Talk extension
- Support for Channel Talk team chat (group) and customer chat (userChat)
- Webhook endpoint for receiving messages
- Function (Command) endpoint for Channel Talk app store integration
- Built-in pairing system using OpenClaw SDK
- Keyword-based bot triggering in group chats
- Group allowlist for selective bot responses
- Full TypeScript support
- Markdown to Channel Talk blocks conversion
- OpenClaw ChannelPlugin implementation with all required adapters

### Features
- **Messaging**: Send and receive messages in both group and user chats
- **Access Control**: Pairing-based authentication system
- **Flexible Triggering**: Keyword-based triggers for group chats
- **API Integration**: Full Channel Talk REST API v5 support
- **OpenClaw Integration**: Native OpenClaw plugin with standard interfaces
